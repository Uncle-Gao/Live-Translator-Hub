## Επισκόπηση Έργου

Το Live Translator Hub είναι μια **εφαρμογή επιτραπέζιου υπολογιστή Electron + React GUI**, που παρέχει αυτόματη μετάφραση στα κινέζικα για δύο εργαλεία προγραμματισμού AI, το Cursor και το Claude. Μέσω ενός ενοποιημένου πυρήνα χρόνου εκτέλεσης μετάφρασης, διαχειρίζεται την ανάπτυξη μηχανών, τη διαμόρφωση κλειδιών API και τη δημιουργία λεξικών και για τις δύο εφαρμογές-στόχους από μία ενιαία διεπαφή.

Αυτό το έργο αποτελεί μια αναβαθμισμένη έκδοση αρχιτεκτονικής του [Live-Translator-Hub](https://github.com/Uncle-Gao/Live-Translator-Hub) – εξελίσσεται από ένα σενάριο CLI σε ένα GUI με πίνακα κατάστασης και καταγραφή σε πραγματικό χρόνο, συνδυάζοντας τις δυνατότητες μετάφρασης των Cursor και Claude σε μια ενιαία ενοποιημένη πλατφόρμα.

## Αρχιτεκτονική

```
live-translator-ecosystem/          # npm workspaces monorepo
├── packages/
│   ├── desktop-app/                # Electron + React GUI (Live Translator Hub)
│   │   ├── electron/main.js        # Κύρια διεργασία, κανάλια IPC και αποθήκευση διαμόρφωσης
│   │   ├── electron/preload.js     # Γέφυρα επικοινωνίας διεργασίας απόδοσης
│   │   └── src/                    # React 19 + Tailwind v4 + Zustand
│   ├── core/                       # Πυρήνας χρόνου εκτέλεσης μετάφρασης (translator-engine.js)
│   ├── patcher-cursor/             # Patcher εφαρμογής Cursor
│   ├── patcher-claude/             # Patcher εφαρμογής Claude
│   └── dict-generator/             # Δημιουργός λεξικών AI
```

### Χρόνος Εκτέλεσης Μετάφρασης

Το `packages/core/src/translator-engine.js` είναι ο μοναδικός χρόνος εκτέλεσης που εισάγεται στην εφαρμογή-στόχο – καθαρό JavaScript προγράμματος περιήγησης, χωρίς εξαρτήσεις από modules. Οι αρμοδιότητές του περιλαμβάνουν:

- **Αντιστοίχιση λεξικού**: Στατικές λέξεις + μοτίβα κανονικών εκφράσεων
- **Γέφυρα πληρεξουσίου μετάφρασης AI**: Σε περιβάλλον Webview, χρησιμοποιεί `postMessage` για να προωθήσει αιτήματα μετάφρασης στο κύριο παράθυρο, παρακάμπτοντας τους περιορισμούς δικτύου CSP
- **Προσωρινή αποθήκευση μετάφρασης**: Μόνιμη προσωρινή αποθήκευση βάσει `localStorage`, με όνομα κλειδιού `live_i18n_cache_<entity_name>`
- **Αναζήτηση ένθετου λεξικού**: Υποστηρίζει λειτουργία `enableNestedDict`

## Κύρια Χαρακτηριστικά

### Ενοποιημένη Διαχείριση Δύο Μηχανών

Διαχειριστείτε την κατάσταση ανάπτυξης μετάφρασης, την έκδοση λεξικού και τους κανόνες αποκλεισμού για τα Cursor και Claude από την ίδια διεπαφή, χωρίς να χρειάζεται εναλλαγή εργαλείων.

### Διείσδυση Webview σε Όλα τα Σενάρια

Μέσω της αρχιτεκτονικής Translation Bridge, η δυνατότητα μετάφρασης AI μπορεί να διεισδύσει από το κύριο παράθυρο σε όλα τα επίπεδα προσθηκών Webview (π.χ. Claude Code), λύνοντας το πρόβλημα αποκλεισμού δικτύου υπό αυστηρές πολιτικές CSP.

### Διάταξη Τεσσάρων Πινάκων Λειτουργιών

| Πίνακας | Λειτουργία |
| :--- | :--- |
| **Cursor Engine** | Ανάπτυξη/Επαναφορά μετάφρασης Cursor, διαχείριση κανόνων αποκλεισμού ανά τομέα για το κύριο παράθυρο και προσθήκες Webview |
| **Claude Engine** | Ανάπτυξη/Επαναφορά μετάφρασης Claude, διαμόρφωση κανόνων παράκαμψης |
| **API Keys** | Διαχείριση κλειδιών API για πολλαπλές μηχανές μετάφρασης AI (υποστηρίζει OpenAI, Anthropic, Google Gemini, DeepL), τα κλειδιά αποθηκεύονται κρυπτογραφημένα μέσω Electron `safeStorage` |
| **Dict Generator** | Εξαγωγή συμβολοσειρών UI από τον πηγαίο κώδικα της εφαρμογής-στόχου και μαζική δημιουργία λεξικών μετάφρασης μέσω AI |

### Διαδραστική Αποσφαλμάτωση

- `Cmd + Option + Shift + B` (Mac) / `Ctrl + Alt + Shift + B` (Win) για εναλλαγή μπλε διακεκομμένου περιγράμματος επισήμανσης
- Στη λειτουργία επισήμανσης, κρατήστε πατημένο το `Option` (Mac) / `Alt` (Win) και τοποθετήστε τον δείκτη πάνω από κινεζικό κείμενο για να δείτε το πρωτότυπο

### Κανόνες Αποκλεισμού ανά Τομέα

Κάθε οντότητα (κύριο παράθυρο και κάθε πρόσθετο) διαθέτει πλήρως ανεξάρτητα σύνολα κανόνων αποκλεισμού (επιλογείς CSS, αντιστοίχιση URL, αντιστοίχιση τίτλου), διασφαλίζοντας ότι οι περιοχές κώδικα και οι βασικές περιοχές αλληλεπίδρασης δεν επηρεάζονται από τη μετάφραση.

### Αυτόματες Ενημερώσεις

Ενσωματωμένο `electron-updater`, υποστηρίζει αυτόματο έλεγχο, λήψη και εγκατάσταση ενημερώσεων εντός εφαρμογής σε macOS.

## Γρήγορη Έναρξη

```bash
# Εγκατάσταση εξαρτήσεων
npm install

# Εκκίνηση λειτουργίας ανάπτυξης GUI
npm run dev

# Δημιουργία διανεμόμενης έκδοσης macOS
npm run build -w desktop-app
```

### Διαδικασία Χρήσης

1. Διαμορφώστε τα κλειδιά μηχανής AI στον πίνακα **API Keys**
2. Μεταβείτε στον πίνακα **Cursor Engine** ή **Claude Engine**
3. Κάντε κλικ στο **Deploy** για αυτόματη ανάπτυξη μετάφρασης
4. Επανεκκινήστε την εφαρμογή-στόχο για να τεθεί σε ισχύ

### Απαιτήσεις Συστήματος

- macOS 13+ (συνιστάται)
- Node.js 18+
- Εγκατεστημένη η επιτραπέζια εφαρμογή Cursor ή Claude

## Ασφάλεια

- **Κρυπτογραφημένη αποθήκευση κλειδιών API**: Αποθηκεύονται κρυπτογραφημένα μέσω Electron `safeStorage` στο `~/.live_translator_hub/api_keys.enc`, χωρίς εγγραφή σε αρχεία διαμόρφωσης
- **Άμεση επικοινωνία**: Τα αιτήματα μετάφρασης πηγαίνουν απευθείας στο API του παρόχου AI, χωρίς ενδιάμεσους διακομιστές
- **Απομόνωση ανά τομέα**: Οι κανόνες αποκλεισμού δεν αγγίζουν τα αρχεία πηγαίου κώδικα
