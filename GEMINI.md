# GEMINI.md

Canonical development instructions live in [AGENTS.md](./AGENTS.md). Keep this file only as a compatibility entry point for Gemini-based tooling.

Important naming note: `claude` in this codebase refers to the supported Claude Desktop target application. Do not rename `packages/patcher-claude`, `claude:*` IPC handlers, or `config.claude` to Codex unless the product scope changes to support Codex Desktop as a target app.
